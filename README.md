# COLOR RUSH X
Test your speed, reflexes, and focus in this addictive arcade challenge by VedGames Production.

Tap the glowing balls before time runs out and climb through ranks from Beginner to Speed God. The game gets faster every few seconds, pushing your reaction time to the limit.

🔥 Features:
• Smooth neon animations
• Reflex-based addictive gameplay
• Rank system (Beginner → Speed God)
• Increasing speed difficulty
• High score save system
• Clean futuristic UI

Can you become the ultimate Speed God? 🚀
